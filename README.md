# TrafficTwin Bengaluru

hosted on streamlit: https://traffictwinbengaluru.streamlit.app/
## Problem

Traffic incidents create unpredictable congestion and inefficient resource allocation.

## Solution

TrafficTwin predicts incident impact and recommends proactive traffic management actions.

## Features

* Risk Intelligence Dashboard
* Incident Impact Prediction
* Similar Incident Explorer
* Traffic Analytics
* Command Center
* Interactive Risk Map
